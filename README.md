# GNUClient (Forge recode)

Minecraft 1.8.9 Forge client mod — Essential Loom + mixins + Java ClickGUI.

## Build

```bash
./gradlew build
```

Output jar: `build/libs/gnuclient-1.0.0.jar` — copy into your mods folder.

## Dev client

```bash
./gradlew runClient
```
